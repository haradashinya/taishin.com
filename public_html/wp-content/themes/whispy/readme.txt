A clean, green SEO-optimized theme.

== Description ==

Whispy is a sharp, lime-ish theme resting on the solid bones of Twenty Ten. Tweaked for improved SEO performance, it features an optional three-column template, optional no-header formatting and the ability to specify title and description text colour, as well as a header widget.

== Installation ==

Upload whispy to /wp-content/themes or you can install the theme through the WordPress installer under **Themes -> Install themes** by searching for it.

== Frequently Asked Questions ==

= Where can I get support for the theme? =
Support is available at the at the Clash Media <a href="http://clashmedia.com/support/">support</a> page.

= How do I get a three-column page? =

When you create or edit a page, simply choose the three-column template option.

== Can I use the 3-column layout on posts? ==

Since 1.5, we've added controls to set blog post index pages and blog posts pages to either 2- or 3-column. Go to 'Appearance->Whispy options' or to the theme customization interface (at /wp-admin/customize.php).
